# Introdução
1.Você tera que instalar o [XAMPP](https://www.apachefriends.org/pt_br/index.html).<br>
2.Instale tambem o [HeidiSQL](https://www.heidisql.com/download.php#google_vignette).<br>
3.Agora ira baixar os arquivos do projeto-Crud.<br>
4.Após ter instalado o XAMPP sera criado uma pasta chamada htdocs, agora com os arquivos baixado você ira criar uma pasta no C:\xampp\htdocs\ crie uma pasta dentro do htdocs.<br>
5.Inicie o XAMPP e ligue o Apache e MySql clicando no botão escrito Start.<br>
6.Inicie o HeidiSQL e use o arquivo de database e o execute-o no HeidiSQL.<br>
7.Então você pode ir no seu navegador e digitar na barra de pesquisa localhost/nomepasta ou 127.0.0.1/nomepasta (pasta criada no passo 3)<br>
8.Finalizado e pronto para uso
